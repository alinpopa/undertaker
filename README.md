An actor based REST api workflows manager

#### How to

##### Prerequisites

- JVM (> 1.8)
- SBT (> 0.13.x)

##### Config

- configs are placed within `src/main/resources/application.conf`

##### Tests

- `sbt test`

##### Run

- `sbt run`
- or
- `sbt assembly && java -jar target/scala-2.11/undertaker-1.0.0.jar`
